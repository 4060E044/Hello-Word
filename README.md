#include <iostream>
using namespace std;
int main()
{
    cout << "Hello, world!" << endl;
    return 0;
}
<iostream>是屬於C++的函式庫
namespace :防止特別在程式碼基底包含多個程式庫時可能會發生的名稱衝突。
std :標準函數庫。
cout也可以使用endl(end of line)控制碼，能做換行的動作
*****************************************
#include <stdio.h>
 
int main()
{

    printf("Hello, World! \n");
 
    return 0;
}
